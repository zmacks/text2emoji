# text2emoji Project Documentation

## Overview
This is a text2emoji conversion project that transforms text content into emoji representations.

## Project Structure
- `app/`: Main application code
  - `main.py`: Entry point for the application
  - `utils.py`: Utility functions
  - `prompts/`: Prompt fragments for Gemini integration
- `README.md`: Project overview and usage instructions
- `pyproject.toml`: Project configuration and dependencies

## Key Features
- Text to emoji conversion
- Modular prompt-based architecture
- Integration with Gemini API

## Development
This project uses Python with virtual environment management. Install dependencies and run the application via the main entry point.

## Agents
The project is configured with three specialized agents:
1. **Doc Writer Agent**: Generates and maintains project documentation
2. **Game Developer Agent**: Assists with game-related development tasks
3. **Play Tester Agent**: Tests and validates gameplay and user experience
